# 个人健康管理系统 java597

## 项目概述

个人健康管理系统是一款集成了智能问诊和智能周报月报功能的系统。通过使用Springboot、Springmvc和vue技术栈，为用户提供了一个易于使用、功能强大的健康管理和问诊平台。

## 技术栈

- 后端：Springboot、Springmvc
- 前端：vue
- 开发环境：Java 1.8、Maven 3.8.1、Node.js 14.16.0
- 数据库：MySQL
- 开发工具：前端-VSCode，后端-IDEA（可选）

## 系统角色

系统定义了以下两种角色：

- 用户：可使用智能问诊、查看周报月报等功能。
- 管理员：负责系统管理，具有普通用户的所有功能权限。

## 功能模块

- 智能问诊：接入deepseek智能问诊接口，提供专业的问题咨询。
- 周报月报：定期生成用户的健康报告。
- 论坛模块：采用协同过滤算法推荐相关论坛内容。

## 运行环境

- Java 1.8 或更高版本
- Maven 3.8.1 或更高版本
- Node.js 14.16.0 或更高版本
- MySQL 数据库

## 部署步骤

1. 环境配置：确保所有依赖的软件和工具正确安装和配置。
2. 数据库设置：导入MySQL数据库结构和数据。
3. 后端启动：通过maven构建项目，并在IDEA中运行启动类。
4. 前端启动：在VSCode或其他前端开发工具中运行前端项目。
5. 系统访问：打开浏览器，输入前端项目部署的地址访问系统。

## 目录结构

```
- 项目根目录
  - frontend：前端项目目录
  - backend：后端项目目录
    - src：后端源代码
    - pom.xml：后端项目依赖管理
  - database：数据库相关文件
```

## 核心亮点

- 集成deepseek智能问诊接口，提供专业的健康咨询服务。
- 定期生成个性化的健康报告，帮助用户更好地掌握健康动态。
- 论坛模块引入协同过滤算法，实现精准内容推荐。
- 代码结构完整，易于理解和维护，确保配置成功的环境可以顺利运行。
